# Neon Island Run 3D

Neon Island Run 3D is a browser-based neon island arcade concept created for portfolio and GitHub Pages.

## Live Demo

https://laurandreea10.github.io/Neon-Island-Run-3D/

## Description

The project presents a futuristic island scene with a glowing path, neon atmosphere, collectible-style visual elements and a responsive layout.

The repository now includes an index.html file, so GitHub Pages has a valid entry point. This fixes the previous 404 once the Pages deployment finishes.

## Highlights

- Neon island visual theme
- Arcade-inspired interface
- Responsive design
- GitHub Pages ready
- Portfolio-friendly presentation
- Reduced-motion friendly decorative animation

## Built With

- HTML5
- CSS3
- GitHub Pages

## Local Usage

Clone the repository and open index.html in a browser.
